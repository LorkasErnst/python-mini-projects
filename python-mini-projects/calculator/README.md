## Taschenrechner

Ein einfacher Taschenrechner, der grundlegende Rechenoperationen durchführt.

### Voraussetzungen
- Python 3.x muss installiert sein.

### Schritte
1. Projekt klonen:
   ```sh
   git clone https://github.com/LorkasErnst/calculator.git
   cd calculator
   ```
2. Programm ausführen:
   ```sh
   python calculator.py
   ```
