## Kalender

Zeigt einen Monatskalender für ein bestimmtes Jahr an.

### Voraussetzungen
- Python 3.x muss installiert sein.

### Schritte
1. Projekt klonen:
   ```sh
   git clone https://github.com/LorkasErnst/calender.git
   cd calender
   ```
2. Programm ausführen:
   ```sh
   python simple_calender.py
   ```
